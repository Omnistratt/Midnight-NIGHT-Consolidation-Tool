Lace Wallet Instructions:

  1. Open Lace wallet extension
  2. Click on your wallet name → Account menu
  3. Select Sign message
  4. Paste the message shown above
  5. Select the payment address (NOT staking address)
  6. Click Sign
  7. Copy the signature and paste below

Eternl Wallet Instructions:

  1. Open Eternl wallet
  2. Go to Settings → Tools
  3. Select Sign message
  4. Paste the message shown above
  5. Choose the payment address to sign with
  6. Click Sign
  7. Copy the signature and paste below

Nami Wallet Instructions:

  1. Open Nami wallet
  2. Access the dApp connection or developer menu
  3. Use Sign Data feature
  4. Paste the message shown above
  5. Sign with payment address
  6. Copy the signature and paste below
